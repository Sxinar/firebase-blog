# Product Context

## Proje Vizyonu
"Semih BAŞARAN" kişisel blog projesi. Kullanıcının düşüncelerini, yazılarını ve projelerini paylaşabileceği; modern, şık (karanlık mod odaklı), hızlı ve tam yönetim paneline sahip bir web uygulaması.

## Temel Özellikler
- **Kişiselleştirilmiş Tasarım:** Shadcn/UI ve Tailwind CSS ile hazırlanmış "Musty Dark" premium tema.
- **Dinamik İçerik Yönetimi:** Firebase Firestore tabanlı yazı ekleme, düzenleme, silme.
- **Admin Paneli:** Sadece yetkili (giriş yapmış) kullanıcıların erişebildiği güvenli yönetim arayüzü.
- **Etkileşim:** Ziyaretçilerin yazılara yorum yapabilmesi.

## Hedef Kitle
Yazılım geliştiriciler, teknoloji meraklıları ve blogun takipçileri.
